# Database schema

Tables principales:

- `users_user`: utilisateurs et roles.
- `suppliers_supplier`: fournisseurs.
- `products_category`: categories.
- `products_product`: articles, seuils et stock courant.
- `purchases_purchaserequest`: demandes d'achat.
- `purchases_purchaserequestline`: lignes de demande.
- `purchases_validationlog`: journal de validation.
- `purchases_purchaseorder`: bons de commande.
- `purchases_purchaseorderline`: lignes de commande.
- `purchases_reception`: receptions.
- `purchases_receptionline`: lignes de reception.
- `stock_stockmovement`: mouvements de stock.
- `purchases_notification`: notifications.
- `purchases_auditlog`: audit transverse.
- `purchases_supplierrequestconversation`: conversations fournisseur.
- `purchases_supplierrequestmessage`: messages.
