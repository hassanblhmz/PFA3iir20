# User manual

## Admin

- Se connecter avec un compte administrateur.
- Ouvrir `Comptes` pour consulter les utilisateurs.
- Utiliser Django Admin pour modifier les roles si besoin.

## Demandeur

- Ouvrir `Demandes`.
- Creer une demande avec une premiere ligne.
- Ajouter des lignes tant que la demande est en brouillon.
- Soumettre la demande.

## Validateur

- Ouvrir les demandes soumises.
- Valider ou rejeter avec commentaire.
- Verifier le journal de validation.

## Acheteur

- Maintenir les fournisseurs et articles.
- Transformer une demande validee en commande.
- Envoyer la commande au fournisseur.

## Fournisseur

- Ouvrir les commandes recues.
- Soumettre un devis, confirmer ou refuser.
- Mettre a jour le statut livraison.

## Magasinier

- Ouvrir une commande envoyee/confirmee.
- Enregistrer une reception.
- Verifier les mouvements dans `Stock`.

## Direction

- Consulter le dashboard, les indicateurs et les alertes stock.
